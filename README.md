# 30 Days of Learning Java

A personal challenge to learn Java through consistent daily practice while building strong programming fundamentals.

## About

This repository documents my 30-day Java learning journey. Each day I learn new concepts, write code by hand, and focus on understanding how programs work rather than simply copying solutions.

I'm using:

- Java
- Maven
- Git
- GitHub
- VS Code

The goal is to build a solid foundation in programming and software development through daily practice.

---

## Goals

- Learn Java fundamentals
- Develop problem-solving skills
- Understand object-oriented programming
- Learn Maven project management
- Become comfortable with Git and GitHub
- Build consistent coding habits

## Projects

### Day 1 — PersonalInfo

Concepts learned:

- Packages
- Imports
- Variables
- Data types (`String`, `int`)
- User input with `Scanner`
- Output with `System.out.print()` and `System.out.println()`

Program features:

- Asks for a user's name
- Asks for their age
- Asks for their favorite color
- Displays the information back to the user

**Find**: `src/main/java/com/learning/PersonalInfo.java`

---

### Day 2 — TemperatureConverter

Concepts learned:

- Comparison operators
- Conditional logic
- `if`, `else if`, and `else`
- Basic mathematical formulas

Program features:

- Converts Celsius to Fahrenheit
- Categorizes temperature as cold, mild, or hot

**Find**: `src/main/java/com/learning/TemperatureConverter.java`

---

### Day 3 — GradeCalculator

Concepts learned:

- `for` loops
- Accumulators
- Repeated user input
- Average calculations
- Multi-branch conditional logic

Program features:

- Accepts multiple student grades
- Calculates class average
- Assigns a letter grade based on the average

**Find**: `src/main/java/com/learning/GradeCalculator.java`
